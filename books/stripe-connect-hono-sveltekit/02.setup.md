# セットアップとモノレポ初期化

この章では、まず動く開発土台を作ります。

章対応コミット:

- ワークスペース初期化: https://github.com/yuu19/stripe-connect-hono-sveltekit-demo/commit/f39c2ffff23e4a88c8bb9f3d0262e0bab3cfc8e3
- ドキュメント/環境テンプレート: https://github.com/yuu19/stripe-connect-hono-sveltekit-demo/commit/685e82babee6e72807e9a1ff593aa73914f80ac8

## 前提ツール

- Node.js 22+
- pnpm 10+
- Wrangler 4+
- Stripe CLI

## リポジトリ作成

```bash
gh repo create stripe-connect-hono-sveltekit-demo --public --clone
cd stripe-connect-hono-sveltekit-demo
```

## 依存インストール

```bash
pnpm install
```

## ローカル起動

```bash
pnpm dev
```

- API Worker: `http://127.0.0.1:8787`
- Web Worker: `http://127.0.0.1:8788`

## 環境変数ファイル

```bash
cp apps/api/.dev.vars.example apps/api/.dev.vars
cp apps/web/.dev.vars.example apps/web/.dev.vars
cp apps/web/.env.example apps/web/.env
```

## Webhook転送

```bash
stripe listen --forward-to http://127.0.0.1:8787/webhooks/stripe
```

ここまでで、以降の章の実装を載せる土台が整います。
